vcpkg_install_make

The latest version of this document lives in the vcpkg repo.

Build and install a make project.

Usage:

vcpkg_install_make(...)

Parameters:

See vcpkg_build_make().

Notes:

This command transparently forwards to vcpkg_build_make(), adding ENABLE_INSTALL

Examples

  • x264
  • tcl
  • freexl
  • libosip2
