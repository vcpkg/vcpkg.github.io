sail

Version: 0.9.1
Port-Version:
Description: The missing small and fast image decoding library for humans (not for machines)
License: MIT
Last Updated: 2024-02-23
Homepage: https://github.com/HappySeaFox/sail
Documentation: N/A
Supported Platforms/Architectures: arm-uwp, arm64-windows, x64-linux, x64-osx, x64-uwp, x64-windows, x64-windows-static, x86-windows


Dependencies

  • vcpkg-cmake (host: true)
  • vcpkg-cmake-config (host: true)


Installation Instructions

Classic Mode:

 vcpkg install sail

Manifest Mode:

 vcpkg add port sail

Features

>all - Enable all codecs
