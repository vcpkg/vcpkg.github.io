osg

Version: 3.6.5
Port-Version: 22
Description: The OpenSceneGraph is an open source high performance 3D graphics toolkit.
License:
Last Updated: 2024-02-23
Homepage: https://www.openscenegraph.com/
Documentation: N/A
Supported Platforms/Architectures: arm-uwp, arm64-windows, x64-linux, x64-osx, x64-uwp, x64-windows, x64-windows-static, x86-windows


Dependencies

  • expat (platform: windows)
  • opengl-registry
  • openimageio (platform: osx)
  • vcpkg-cmake (host: true)
  • vcpkg-cmake-config (host: true)
  • zlib


Installation Instructions

Classic Mode:

 vcpkg install osg

Manifest Mode:

 vcpkg add port osg

Features

>collada - Support for Collada (.dae) files
Dependencies:
  • collada-dom
>default-features - Platform-dependent default features
Dependencies:
  • osg [openexr] (default-features: false) (platform: !uwp)
  • osg [rest-http-device] (default-features: false) (platform: !windows)
  • osg [fontconfig] (default-features: false) (platform: !android & !ios & !uwp)
  • osg [nvtt] (default-features: false) (platform: !x86 & !(windows & arm64) & !uwp)
>docs - Build OpenSceneGraph reference documentation using doxygen (use: make doc_openscenegraph doc_openthreads
No Dependencies
>examples - Enable to build OSG Examples
Dependencies:
  • libiconv (platform: windows)
  • osg [freetype, plugins] (default-features: false)
  • sdl2
>fontconfig - Enable Fontconfig support for osgText
Dependencies:
  • fontconfig
  • osg [freetype] (default-features: false)
>freetype - Enable Freetype support
Dependencies:
  • freetype (default-features: false)
>nvtt - Build texture processing tools plugin
Dependencies:
  • nvtt
>openexr - Build the exr plugin
Dependencies:
  • openexr
>packages - Set to ON to generate CPack configuration files and packaging targets
No Dependencies
>plugins - Build most OSG plugins
Dependencies:
  • curl (default-features: false)
  • gdal (default-features: false)
  • giflib (platform: windows)
  • jasper (default-features: false)
  • libgta
  • libiconv (platform: windows)
  • libjpeg-turbo
  • libpng
  • libxml2 (platform: windows)
  • tiff (default-features: false)
>rest-http-device - Build RestHttpDevice plugin
Dependencies:

  • boost-asio
  • boost-bind
  • boost-core
  • boost-lexical-cast
  • boost-logic
  • boost-smart-ptr
  • boost-tuple
>sdl1 - Build SDL 1 plugin, and enable SDL 1 app features
Dependencies:
  • sdl1
>tools - Enable to build OSG Applications (e.g. osgviewer)
Dependencies:
  • curl (default-features: false)
  • libiconv (platform: windows)
  • libjpeg-turbo
  • libpng
  • osg [freetype, plugins] (default-features: false)
