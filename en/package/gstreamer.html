gstreamer

Version: 1.22.5
Port-Version: 7
Description: GStreamer open-source multimedia framework core library
License: LGPL-2.0-only
Last Updated: 2024-02-23
Homepage: https://gstreamer.freedesktop.org/
Documentation: N/A
Supported Platforms/Architectures: arm-uwp, arm64-windows, x64-linux, x64-osx, x64-uwp, x64-windows, x86-windows


Dependencies

  • glib
  • glib (host: true)
  • opengl (platform: windows | osx)
  • vcpkg-tool-meson (host: true)


Installation Instructions

Classic Mode:

 vcpkg install gstreamer

Manifest Mode:

 vcpkg add port gstreamer
