simage

Version: 1.8.3
Port-Version:
Description: Image file format library abstraction layer
License: ISC
Last Updated: 2024-02-23
Homepage: https://github.com/coin3d/simage
Documentation: N/A
Supported Platforms/Architectures: arm-uwp, arm64-windows, x64-linux, x64-osx, x64-uwp, x64-windows, x64-windows-static, x86-windows


Dependencies

  • vcpkg-cmake (host: true)
  • vcpkg-cmake-config (host: true)


Installation Instructions

Classic Mode:

 vcpkg install simage

Manifest Mode:

 vcpkg add port simage
